# About This Site

This is a modern static website built with cutting-edge web development tools, showcasing the power of combining **Hugo** with **Vite** and **Tailwind CSS 4**.

## Technology Stack

### Hugo
- **Static Site Generator**: One of the fastest SSGs available
- **Content Management**: Markdown-based content with front matter
- **Templates**: Powerful Go template system for layouts
- **Multilingual**: Built-in internationalization support

### Vite
- **Build Tool**: Modern, fast, and lightweight
- **Dev Server**: Hot Module Replacement for instant updates
- **Optimization**: Code splitting and tree shaking
- **Ecosystem**: Rich plugin system

### Tailwind CSS 4
- **Utility-First CSS**: Rapid UI development
- **Vite Integration**: Native Vite plugin for seamless experience
- **Responsive Design**: Mobile-first approach
- **Customization**: Highly configurable design system

## Development Workflow

The development workflow combines the strengths of all three tools:

1. **Content Creation**: Use Hugo's `new content` command
2. **Styling**: Write Tailwind classes with live reload
3. **Asset Building**: Vite processes and optimizes CSS
4. **Static Generation**: Hugo builds the final static site

## Commands

```bash
# Install dependencies
pnpm install

# Development with hot reload
pnpm dev:full

# Build CSS only
pnpm build

# Build complete site
pnpm build:full
```
